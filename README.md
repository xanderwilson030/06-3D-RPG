# 06-3D-RPG
Project 6 for MSCH-C220, 6 December 2021

An exploration of Godot in 3D.

## Implementation
Built using Godot 3.3.3

Extra Features:
- Custom Skybox
- Graphical details (Reflections, SSAO, Fog)
- PBR Textures
- Weapon Shooting Noise
- Background Music
- Main Menu (Juicy)
- Win Screen (Juicy)
